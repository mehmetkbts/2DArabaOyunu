# 2D Sonsuz Araba Oyunu/2D Endless Car Game
Bu proje, Unity kullanılarak yapılmış basit bir 2D araba oyunudur. Oyuncu, bir arabayı sonsuz bir yolda kontrol ederek engellerden kaçmaya çalışır. Oyun, temel mekaniklere odaklanmıştır ve kullanıcı arayüzü (menü sistemleri) içermemektedir.
This is a simple 2D car game built with Unity. The player controls a car driving on an endless road while avoiding obstacles. The project focuses on core gameplay mechanics and **does not include any UI menus.

## Özellikler/Features
- 2D üstten görünüşlü araba kontrolü/ 2D top-down car control
- Sonsuz olarak ilerleyen yol sistemi/Endless scrolling road system
- Engellerle çarpışma mekaniği/ Obstacle collision detection
- Basit oyun fiziği ve kontrolle/Basic physics and controls

## Kontroller/Controls
W= İleri gitme/forward
A= Sola gitme/Left
D= Sağ gitme/Right
